Retail Sales Report

This sales report includes two dashboards that provide a comprehensive and detailed analysis of the sales performance. In order to ensure the accuracy and completeness of the data, I performed a series of data transformation steps using the Power Query Editor. These steps included converting data types, adding new columns, and so on. The report is based on five tables, which were linked together through a data model to capture the relationships between different tables. To further enhance the report, calculated measures and columns were created using DAX, which allowed for the creation of more informative and visually appealing visualizations. These measures and columns were designed to provide key insights into the sales performance, enabling users to quickly identify trends, patterns, and other important information.

HR Report

This HR report comprises of two dashboards that provide valuable insights into the HR data. The visualizations were designed in a user-friendly way to enable decision-makers to quickly and easily access a breakdown of workers' information, allowing them to make swift decisions based on the visualizations. To ensure data accuracy and completeness, several data transformation steps were carried out using the Power Query Editor. These steps included data type conversion, dropping irrelevant columns, the creation of new columns, and so on. Various calculated measures were created using DAX to enhance the visualizations and provide a more informative and visually appealing experience for users. The report is based on two tables that were linked together through a data model that captures their relationships.
